# 🐍 Snake Game Pro

An advanced and interactive **Snake Game** built using **HTML, CSS, and JavaScript**.
This version includes smooth gameplay, increasing difficulty, high score tracking, and modern UI.

---

## 🚀 Features

* 🎮 Classic Snake Gameplay
* ⚡ Speed increases as score grows
* ⏸️ Pause / Resume (Press Spacebar)
* 🏆 High Score saved using Local Storage
* 🍎 Smart food spawning (never overlaps snake)
* 🎨 Grid-based clean UI
* 🔄 Restart button

---

## 🛠️ Tech Stack

* HTML5 (Canvas)
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Play

* ⬆️ Arrow Up → Move Up
* ⬇️ Arrow Down → Move Down
* ⬅️ Arrow Left → Move Left
* ➡️ Arrow Right → Move Right
* ⏸️ Press **Space** to Pause/Resume

---


## ⚙️ How It Works

* Snake moves continuously on a grid
* Eating food increases score and speed
* Game ends if snake hits wall or itself
* High score is stored in browser using Local Storage
